# Femto-Lite v1.0.0 (Updated 02APR26 APM)

**Written by:** Adam Micolich

**Purpose:** Design aspects for a low-cost USB-C powered two-channel transimpedance amplifier with two gain settings (1000V/A and 10000V/A).

## Structure

The top level has three folders -- *CNC Parts*, *3D Print Parts* and *PCBs & Schematics*.

The *CNC Parts* folder contains CNC-ready files for producing all the panels for the Femto-lite box in metal.

The *3D Print Parts* folder contains 3D print-ready files for producing all the panels for the Femto-lite box in plastic.

The *PCBs & Schematics* folder contains the PCB design, Schematic and the Gerber file set for production for the PCB used in this project. It also contains a parts list for all the additional components required, along with some photographs and test data.

Each of the three folders above has its own `README.md` file with further details regarding the contents of those folders.
